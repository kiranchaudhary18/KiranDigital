## ğŸš¦ Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn or pnpm

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/kiranchaudhary18/portfolio-nextjs.git
   cd portfolio-nextjs
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

## ğŸ¨ Customization

### Personal Information

1. **Update metadata in `app/layout.tsx`**:
   - Change title, description, keywords
   - Update Open Graph images and URLs
   - Modify social media handles

2. **Update content in components**:
   - Edit personal information in `Hero.tsx`
   - Update skills and technologies in `Skills.tsx`
   - Add your projects in `Projects.tsx`
   - Update certificates in `Certificates.tsx`
   - Modify contact information in `Contact.tsx`

### Styling

1. **Colors**: Update the color palette in `tailwind.config.js`
2. **Fonts**: Modify fonts in `app/layout.tsx` and `globals.css`
3. **Animations**: Customize animations in `globals.css`

### SEO Configuration

1. **Domain**: Update the base URL in:
   - `app/layout.tsx` (metadataBase)
   - `app/sitemap.ts`
   - `app/robots.ts`

2. **Social Media Images**: Add your Open Graph images to `/public/`
   - `og-image.jpg` (1200x630)
   - `twitter-image.jpg` (1200x675)
